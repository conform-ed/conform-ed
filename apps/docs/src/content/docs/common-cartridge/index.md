title: Common Cartridge
description: Read and write IMS Common Cartridge packages with @conform-ed/common-cartridge.
---

`@conform-ed/common-cartridge` works with [IMS Common Cartridge](https://www.imsglobal.org/cc/)
packages — the `imsmanifest.xml`, its resources, web content, web links, and the other cartridge
surfaces — for both v1.3 and v1.4.

```bash
bun add @conform-ed/common-cartridge
```

It pairs with the Common Cartridge schemas in [`@conform-ed/contracts`](/contracts/)
(`@conform-ed/contracts/common-cartridge/v1_3` and `…/v1_4`).

API guides and worked import/export examples are being expanded in this section.
